# Text-Glow Generator

Welcome to Text-Glow, a dynamic web application that allows you to create beautiful, glowing text with customizable colors. This tool is perfect for generating eye-catching headers, logos, or snippets for your projects.

# Features
This project is built with a focus on user experience and efficiency, providing a wide range of features to make text styling easy and fun.

*   **Live Preview:** See your glowing text creations in real-time as you type.
*   **Custom Color Selection:** Use an interactive color wheel to pick the exact glow color you want.
*   **Multi-Color Text:** Combine multiple text segments with different colors in the same preview.
*   **Automatic Spacing:** A smart spacing feature automatically adds a space between new text segments, which can be toggled on or off.
*   **Efficient Workflow:** A "Clear" button and numerous keyboard shortcuts help you work faster.
*   **Informative Guide:** An integrated info panel provides a quick reference for all available shortcuts.

# ⌨️ Keyboard Shortcuts
To enhance productivity, the application supports a variety of keyboard shortcuts:

| Shortcut            | Action                                      |
| ------------------- | ------------------------------------------- |
| `Enter`             | Apply the glow effect to the current text.  |
| `Ctrl + Backspace`  | Clear all text from the input and preview.  |
| `Ctrl + I`          | Toggle the shortcut information panel.      |
| `Ctrl + J`          | Toggle automatic spacing between segments.  |
| `Ctrl + 1` to `0`   | Select from 10 preset colors (Red, Blue, Green, White, etc.). |
