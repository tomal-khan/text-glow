Keyboard Shortcuts

  • Enter: Apply glow to the current text.
  • Ctrl + Backspace: Clear all text from input and preview.
  • Ctrl + I: opens information page.
  • Ctrl + J: disables and enables space before text.
  • Ctrl + 1 Select a preset color (red)
  • Ctrl + 2 Select a preset color (blue)
  • Ctrl + 3 Select a preset color (green)
  • Ctrl + 4 Select a preset color (yellow)
  • Ctrl + 5 Select a preset color (cyan)
  • Ctrl + 6 Select a preset color (orange)
  • Ctrl + 7 Select a preset color (purple)
  • Ctrl + 8 Select a preset color (pink)
  • Ctrl + 9 Select a preset color (black)
  • Ctrl + 0: Select White.
